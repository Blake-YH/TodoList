# Branding Guide

## Product Name

- Primary name: `TodoList`
- Positioning line: `Local-first lightweight desktop todo app`

## Brand Direction

The product should feel:

- Focused
- Calm
- Fast
- Practical

It should avoid:

- Heavy enterprise aesthetics
- Overly playful productivity branding
- Visual clutter

## Core Colors

- Primary amber: `#F3A522`
- Success green: `#3AC28C`
- Info blue: `#6EA8FE`
- Accent pink: `#F472B6`
- Accent violet: `#A78BFA`
- Accent orange: `#F97316`

## Surface Colors

- Main background: deep graphite gradient
- Panel background: translucent dark glass
- Text: warm off-white
- Muted text: blue-gray

## Icon Direction

The icon should communicate:

- A task list
- A completed action
- A lightweight utility feel

Suggested shape language:

- Rounded square container
- Check mark or checklist motif
- Warm amber focal accent
- Clean contrast for Windows taskbar visibility
